# v1.11.0 GitHub Release Notes

## v1.11.0 — Mobile Rich Match Dashboard, Provider Leaders, and Group Race

World Cup 2026 AI Stats v1.11.0 upgrades the self-hosted dashboard from fixture tracking into a richer matchday experience built around stored provider data.

### Highlights

- Added provider-backed rich match detail with match events, cards, substitutions, statistics, formations, lineups, referee, and weather context.
- Added status-first fixture browsing with group filters and a responsive detail panel.
- Added sticky Quick Links navigation for faster section access.
- Added provider-derived top scorer, yellow-card, and red-card leaderboards.
- Added explicit assist-data unavailable behavior when the provider does not supply assist events.
- Added latest completed-match summary backed by stored provider event data.
- Added Group Race showing the top two teams in each populated group.
- Preserved local Ollama summary support with deterministic fallback safeguards.
- Refreshed release, architecture, demo, roadmap, and portfolio documentation.
