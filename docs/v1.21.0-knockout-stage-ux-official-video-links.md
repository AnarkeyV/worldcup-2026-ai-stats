# v1.21.0 — Knockout Stage UX & Official Match Video Links

## Summary

v1.21.0 is a source-only live-product UX release for the tournament knockout phase. It promotes recognised provider-backed knockout fixtures in the dashboard, keeps group-era context accessible through progressive disclosure, repairs desktop More-menu behavior, and strengthens how the product presents manually verified official match-video links.

## Dashboard hierarchy

- Matchday prefers recognised stored knockout fixtures when they exist.
- The supported display order is Round of 32, Round of 16, Quarter-finals, Semi-finals, Third-place Playoff, then Final.
- Later rounds do not appear unless stored provider data supplies those stages.
- Unknown or malformed stage labels are never promoted into the knockout view.
- Existing factual fixture-state rules remain unchanged: time does not infer live status, and unresolved/unknown fixtures remain unavailable unless the established conservative upcoming presentation is justified by stored data.

## Group Stage progressive disclosure

- Group Standings, Group Race, and Group Insights sit inside a native Group Stage disclosure.
- The disclosure collapses only after recognised knockout fixtures are loaded.
- The Groups navigation destination opens and focuses the disclosure.
- Keyboard focus styles and native disclosure behavior are preserved.

## Navigation repair

- The desktop More menu no longer depends on an overflow-clipped navigation context.
- Both desktop and mobile More menus close on destination selection, Escape, outside click/tap, and viewport mode changes.
- Escape restores focus to the corresponding More summary control.

## Official Match Video

- Official Watch remains local-record, outbound-link only.
- `mediacorp_sports_youtube` accepts only a curator-verified direct YouTube video URL associated with the official `@SportsMediacorp` channel.
- There is no YouTube API key, automated discovery, scraping, embedding, downloading, rehosting, or public video-write endpoint.
- Cards distinguish a verified match-specific video from an official coverage hub and from an honest unavailable state.
- No production video record is inserted in this release. See [Official Match Video Curation](official-match-video-curation.md) for the separate evidence and approval gates.
