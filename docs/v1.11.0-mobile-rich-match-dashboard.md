# v1.11.0 — Mobile Rich Match Dashboard, Provider Leaders, and Group Race

## Goal

Make World Cup 2026 AI Stats genuinely useful for live personal match tracking by turning fixture data into a mobile-friendly matchday dashboard with provider-backed detail, transparent player leaders, and qualification context.

---

## Delivered Scope

### Rich Match Detail

- Added persistent provider-backed `MatchDetail` storage.
- Added `GET /fixtures/{fixture_id}/detail`.
- Stored goals, cards, substitutions, formations, lineups, statistics, referee details, and weather context when provided.
- Added dashboard tabs for overview, timeline, statistics, and lineups.

### Fixture Browser

- Added Completed, Live, and Upcoming status controls.
- Added group-level fixture browsing.
- Added selected-fixture state and match-detail focus.
- Added responsive layout with fixture browsing and detail panel.
- Added sticky Quick Links navigation.

### Provider Leaders

- Added `GET /players/leaders`.
- Derived top scorers, yellow-card leaders, and red-card leaders from stored completed match details.
- Added filters by team and group.
- Normalized scorer names so event metadata does not leak into player names.
- Made lack of provider assist data explicit rather than fabricating an assist leaderboard.

### Latest Result

- Added `GET /ai/latest-completed/summary`.
- Summarises the newest completed match from stored provider data.
- Prioritises major incidents when available, followed by scoreline and scorer context.

### Group Race

- Extended `GET /ai/insights` with `group_race`.
- Shows the top two teams in each populated group.
- Added Group Race board to Structured AI Insights.
- Replaced generic strongest-attacks emphasis in structured insight output with group-position context.

## Important Data-Integrity Rules

- Do not show generic sample player records as live provider data.
- Do not invent assist leaders when assist events are absent.
- Use deterministic data views for leaders and Group Race.
- Treat browser output as the visual source of truth when terminal encoding damages accented names.
- Do not run provider sync or backfill unnecessarily on a stable runtime, because it can change data or trigger notifications.
